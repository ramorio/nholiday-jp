# nholiday-jp

## About

日本の祝日を取得します。  
ここでの祝日は、国民の祝日、振替休日、国民の休日を含みます。  
1949年から2099年まで対応しています。

## Usage

2019年の祝日一覧を取得します。

```js
import NHolidayJp from 'nholiday-jp';

const holidays = NHolidayJp.getByYear(2019);
holidays.forEach(v=>
  console.log(`${v.month}月${v.date}日 ${v.name}`)
);
//=> 1月1日 元日

//   5月5日 こどもの日

//   7月15日 海の日
//   8月11日 山の日
```
